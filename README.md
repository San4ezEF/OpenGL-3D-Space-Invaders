# Space Invaders 3D Prototype

Прототип классической аркадной игры Space Invaders, реализованный в трехмерном пространстве с использованием OpenGL.



## Особенности проекта
- **3D Освещение:** Реализована модель освещения Ламберта (диффузное и фоновое) для придания объема кубам и сферам.
- **Плавная анимация:** Все движения (игрок, пули, пришельцы) привязаны к `deltaTime`, что обеспечивает одинаковую скорость на мониторах с разной частотой обновления.
- **Система волн:** После уничтожения всех врагов начинается новая волна с увеличенной скоростью движения.

## Технологии

- **Библиотеки:**
  - `GLFW` — создание окна и обработка ввода.
  - `GLEW` — доступ к функциям современного OpenGL.
  - `GLM` — математика для 3D графики (матрицы, векторы).



## Управление
- **Стрелки Влево/Вправо:** Перемещение игрока.
- **Пробел:** Стрельба.
- **Esc:** Выход из игры.
